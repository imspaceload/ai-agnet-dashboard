# LexiForge AI Dashboard

LexiForge AI is a lightweight starter dashboard for building practical AI agents for company workflows.

## Includes
- Branded landing page with product positioning
- Login flow that reveals the builder dashboard
- Process selection for common agent use cases
- Agent input fields for:
  - System prompt
  - Agent persona
  - Agent description
  - API keys (OpenAI, Voice provider, Vector DB)

## Run locally
Because this is a static app, you can open `index.html` directly or run a simple web server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
